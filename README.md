# Elgreco

About the Spainsh painter from Greece

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'elgreco'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install elgreco

## Usage

in terminal bring type in elgreco
